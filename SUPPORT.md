# Support Policy

This is a personal source-available project. Version 1.1 is the current stable release, but support and compatibility on every system are not guaranteed.

Constructive bug reports and technical feedback are welcome. Abusive, threatening, insulting, or hostile comments may be ignored, deleted, or reported.

This project is shared for people who are interested in experimenting with Forza telemetry-based DualSense haptics. It is not intended to satisfy every controller preference or every system environment.

## Before Reporting An Issue

Please include the following information when possible:

- App version
- Windows version
- Forza version or edition
  - Steam
  - Xbox App
  - Microsoft Store
  - Game Pass
- Controller connection
  - USB
  - Bluetooth
- Whether DSX, DS4Windows, Steam Input, reWASD, or similar tools are running
- Whether Forza Data Out is enabled
- UDP port used by Forza and the app
- Whether `Select DualSense` was completed in the app
- Diagnostic log from the app, if available
- Screenshot or video of the problem, if possible

Issues without enough information may not be answered.

## Not Guaranteed To Work

The following environments are not guaranteed to work:

- Xbox App / Microsoft Store / Game Pass versions
- Bluetooth DualSense haptic output
- DSX / DS4Windows exclusive device access
- Virtual Xbox controller input
- Custom Windows loopback, firewall, or network configurations
- Every DualSense firmware, Windows audio routing setup, or Forza update

## Community Policy

Please keep feedback technical and specific. Reports that include clear reproduction steps are much easier to investigate.

Hostile comments, controller preference arguments, demands for guaranteed support, or unrelated requests may be closed without response.
